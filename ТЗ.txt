Для розробки макету використовую середовище Visual Studio Code. Також, для зручності розробки тут і вдома, буду використовувати GIT. 

Отож, для розробки макету, потрібно як мінімум використати такі технології:
- HTML5;
- CSS3;
- JS.

Для CSS потрібно підключити фреймворки Bootstrap та Tailwind CSS.
Також для слайдеру на макеті необхідно підключити бібліотеку Slick Slider.
